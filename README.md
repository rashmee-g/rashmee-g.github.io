## Featured Projects

### AI-Powered Public Speaking Coach (Capstone Project)
An in-progress mobile application that provides **real-time and post-session feedback** on public speaking using **computer vision, audio analysis, and NLP**.

**Highlights:**
- React Native frontend focused on user flows, feedback visuals, and progress tracking
- Multimodal pipeline using tools like MMPose, DeepFace, Librosa, Speech-to-Text, and Sentence-BERT
- Node.js + Python backend integration via REST APIs
- Embedded scope/design documentation (where applicable)

---

### SyncTime — Google Calendar Social Scheduling (UI/UX Case Study)
A UI/UX design project developed through the BluePrint @ Rutgers fellowship to simplify shared scheduling and help groups find mutual availability without back-and-forth messaging.

**Highlights:**
- User research with 40+ survey responses
- Lo-fi and hi-fi wireframes
- Interactive Figma prototype (embedded)
- Final design presentation embedded on-site

---

### ClassCheck — Geolocation-Based Attendance System (Software Case Study)
A full-stack attendance tracking system for university classrooms using geolocation verification and role-based dashboards.

**Highlights:**
- Node.js backend with 10+ REST APIs
- Firebase Realtime Database (NoSQL schema design)
- Auth0 authentication with MFA
- Attendance reports and exception/appeal workflow
- Embedded project report and schema visuals

---

### Hardware-Integrated Traffic Light System
A sequential logic circuit simulating real traffic signal behavior using digital design principles.

**Highlights:**
- State-based control and timing validation
- Hardware debugging and iteration
- Schematic and final build documentation
